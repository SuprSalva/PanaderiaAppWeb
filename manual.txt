instalar flowbite:

Crea el entorno virtual
    Windows:
    python -m venv env o py -3.11 -m venv env

    macOS:
    python3 -m venv env

Activar el Entorno
    Windows:
    .\env\Scripts\activate
    
    macOS:
    source env/bin/activate

Una vez activo el entorno virtual, ejecuta 
pip install -r requirements.txt

pip freeze > requirements.txt
